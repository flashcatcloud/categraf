# system

系统负载相关的采集插件

## 监控大盘和告警规则

该 README 文件所在的同级目录下提供了以下监控配置文件：

- `dashboard.json`: 夜莺 Dashboard，可以直接导入夜莺使用。
- `dashboard_grafana.json`: Grafana Dashboard，导入后选择 Prometheus 兼容数据源即可使用。
- `alerts-linux.json`: Linux 主机相关告警规则，可以导入夜莺使用。

## Dashboard and alert rules

This directory provides the following monitoring configuration files:

- `dashboard.json`: Nightingale Dashboard, ready to import into Nightingale.
- `dashboard_grafana.json`: Grafana Dashboard. Select a Prometheus-compatible datasource after import.
- `alerts-linux.json`: Linux host alert rules for Nightingale.
